# Document Automation Lab

Small experiments for generating, organizing, and validating business documents used in construction workflows.

## Goal

Reduce repetitive document work while keeping generated files predictable, readable, and easy to review.

## Planned Features

- PDF form generation examples
- Quote request templates
- Contract document experiments
- Printable HTML layouts
- Data-driven document sections

## Tech Stack

- React experiments planned
